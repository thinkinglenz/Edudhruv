# WordPress → Next.js Migration Backup Checklist
Run through these BEFORE switching DNS to Vercel.

---

## ✅ Step 1 — Run the Auto-Migration Script (posts + mock data)
```bash
cd ~/edudhruv-next
pip install requests python-dotenv
python automation/wp_export.py
```
This saves everything to `migration/wp_export.json` and updates localhost instantly.

---

## ✅ Step 2 — Export WordPress Database (leads + all data)
1. Log into cPanel → **phpMyAdmin**
2. Select the `edudhruv_*` database
3. Click **Export** → Format: SQL → Click **Go**
4. Save the `.sql` file somewhere safe
5. Specifically check these tables:
   - `wp_edudhruv_leads` → your student leads
   - `wp_edudhruv_docs` → document uploads
   - `wp_posts` → all blog content (auto-covered by script above)

---

## ✅ Step 3 — Download wp-content/uploads (media files)
Via cPanel File Manager:
1. Go to `/home/edudhruv/domains/edudhruv.com/public_html/wp-content/uploads/`
2. Right-click → **Compress** → Download the zip
3. Also download: `edudhruv-private/` (student document uploads — sensitive!)
